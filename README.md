# Small-Enterprise-Lab-CCNA-

CCNA-level enterprise networking lab featuring VLANs, trunking, inter-VLAN routing, DHCP, and ACL security configuration.
------------------------------------------------------------------------------------------------------------------
## Overview
This project simulates a small enterprise network built in Cisco Packet Tracer. The lab includes VLAN segmentation, trunking, inter-VLAN routing, DHCP, and ACL implementation.

This project was created as part of my networking portfolio and is also intended to help other CCNA students practice and understand enterprise networking concepts.

------------------------------------------------------------------------------------------------------------------

# Devices Used
- 1 Router
- 2 Switches
- 4 PCs
